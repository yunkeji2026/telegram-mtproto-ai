常见问题

你可能关心的问题

部署费 11,999 USDT 具体包含哪些？+

包含现有成熟功能范围内的完整系统私有化部署、TikTok/Facebook/Messenger 引流链路配置、设备集群接入与投屏群控、AI 客服话术与多语言知识库初始化、反风控与代理配置、数据看板与告警接入、端到端联调，以及 1 对 1 运营培训。一次性收取。

月服务费 3,000 USDT 是必须的吗？+

建议保留。社交平台 UI 和风控策略持续变化，月服务费用于保障系统稳定运行、平台适配跟进、反风控调优、版本迭代与优先技术支持。这是自动化系统长期稳定投产的关键保障。

硬件是你们提供还是我们自己买？+

硬件（主机、手机、配件）属于客户一次性资产投入，独立于软件费用。你可以按我们提供的配置清单自行采购，也可委托我们代为采购与整备。由于硬件价格每日波动，本方案只给出配置与数量、不标注价格，实际以采购当日市场价为准。

除了 TikTok 和 Facebook，还能做别的平台吗？+

可以。系统采用统一的多平台动作引擎，Instagram、WhatsApp、Telegram、Twitter/X、LinkedIn、LINE 等均已具备同栈基础能力。若需要将某平台作为正式引流渠道投产，我们按需求做二次开发接入，单独评估报价。

一台电脑能带多少部手机？+

每台工作机通过独立供电的工业 USB HUB 可稳定管理约 15–16 台真机。规模更大时通过增加 Worker 节点横向扩展，单个主控集群可管理 200+ 台设备。

账号会不会被封？+

系统内置完整反风控栈：拟人行为曲线、分平台配额限流、冷却机制、VPN/代理地理伪装、设备指纹隔离、养号期渐进放量等，最大化账号存活率。但社交平台风控客观存在，任何自动化都无法 100% 保证不被风控，因此建议合规操作 + 账号分层运营。

多久能上线出效果？+

标准实施约 7–10 天完成部署、配置与灰度上线。新账号需要约 7 天养号期后才进入正常触达节奏，通常 2–4 周可看到稳定的引流与转化数据，随后进入放量优化阶段。
