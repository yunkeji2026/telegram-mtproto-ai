## 一、深度人设「真人感」系统（7 波，默认可灰度）

核心目标：从「文字机器人」升级为**有生活、记得经历、会主动回指、有脾气、可自我维护**的陪伴人设。

### 代码位置
| 模块 | 作用 |
|---|---|
| `src/companion/deep_persona.py` | 5 层纯核心 + 巩固/召回/回指/漂移守卫/生活分享（确定性、可测） |
| `src/companion/deep_persona_store.py` | 运行期状态 SQLite（`config/deep_persona.db`）：关系画像/内部梗/未收尾话题/经历(含向量)/生活分享节流 |
| `src/companion/deep_persona_stats.py` | 观测计数（巩固/画像/内部梗/经历/回指/生活分享/漂移拦截）→ `/api/workspace/metrics.deep_persona` |
| `src/companion/deep_persona_runtime.py` | 同步 embedder + LLM 注册表（E1 语义召回 / E2 画像精修，避开 async 边界） |
| 注入点 | `ai_client._build_system_instruction`（消费）；`ingest.py`（累积+巩固）；`skill_manager._life_beat_opener`（主动分享） |
| 人设数据 | `config/profiles_runtime.yaml`：每个人设的 `life_arc` / `tastes` / `humor` / `temperament` / `banter_profanity` |

### 分层能力（L1–L5 + 巩固/召回/精修）
- **L1 生活线**：人设自传时间线（`life_arc`，3 天一相位窗，前后一致、可被回指）。
- **L2 关系画像 + 不问就回指**：巩固「我眼中的 TA」，主动提共同往事（偶发闸 + 危机抑制）。
- **L3 口味/立场 + 内部梗**：稳定喜好防「一味附和」；反复出现的梗/昵称自动沉淀复用。
- **L4 经历式记忆**：事件+情感+叙事，情感×时近×相关加权召回（E1 语义向量可选）。
- **L5 拟人细节**：时空锚定（深夜/周一）+ 刻意不完美（熟络阶段概率触发）。
- **巩固/自愈**：机会式节流（15min/会话）巩固画像+内部梗；漂移命中人设硬禁项→拒写+清污自愈。
- **反打扰（D2/E5）**：主动分享频次上限 + 最小间隔 + 静默时段。

### 开关（`config/config.local.yaml → companion.deep_persona`）
```yaml
companion:
  deep_persona:
    enabled: true            # 总闸；false=整块关、行为回到升级前
    life_line: true          # L1（需 persona.life_arc）
    relationship: true       # L2（需 store 数据）
    tastes: true             # L3a（需 persona.tastes）
    inside_jokes: true       # L3b
    experiential: true       # L4
    texture: true            # L5
    callback_probability: 0.35
    imperfection_probability: 0.12
    life_share_max_per_week: 2
    life_share_min_gap_hours: 48
    life_share_quiet_start_hour: 0
    life_share_quiet_end_hour: 8
    llm_refine: false        # E2 画像 LLM 精修（走 deepseek，有 API 成本；默认关）
    semantic_recall: true    # E1 语义向量召回（本机 Ollama，免费；观测见 metrics.deep_persona.embedder）
```
**回退**：`enabled: false` 或删掉整段 → 重启，回到纯文字情绪 + 静态人设。
