# 产品目录与品牌命名方案（Product Catalog & Branding）

> 定位：一家**专注 AI 软件开发**的科技公司，下设「AI 换脸 / AI 声音克隆 / 实时直播换脸换声 / 实时换语言 / 聚合智能聊天获客」五条产品线。
> 总公司品牌：**无界科技 · BOUNDLESS**（替代旧品牌「华灵科技 HuaLing Tech」）。

---

## 一、公司定位（一句话）

> **用 AI 让任何人，以任意面孔、声音、语言，实时沟通并自动成交 —— 无界。**

业务灵魂是 **「无界」**：我们的 AI 部署本身就**没有界限**（无审查、无限制、私有自由部署），而五条产品线正好各自**打破一种边界**。

---

## 二、总公司命名（已锁定）

### 无界科技 · BOUNDLESS

- **中文「无界」**
  - 没有边界、没有限制。既指**技术上**的自由部署（无审查、数据私有、想怎么用就怎么用），也指**产品上**打破容貌/声音/语言/沟通的一切界限。
  - 两字大气、好记、好念，中外通吃。
- **英文「BOUNDLESS」**
  - 全球最易懂的"无界/无限"词，自带格局感（Boundless possibilities）。
  - 品牌口号：**"Communication, Boundless."（让沟通，无界。）**
- **为什么是它**：业务横跨音视频生成与智能对话，需要一个能统领多产品线、又精准命中"无审查自由部署"卖点的名字。"无界"名实相符，不是空话。

### 品牌故事：无界 = 打破五界

| 产品 | 打破的「界」 |
|------|-------------|
| 幻颜（换脸） | 容貌之界 |
| 幻声（克隆声音） | 声音之界 |
| 幻影（实时直播换脸换声） | 身份之界 |
| 通译（实时换语言） | 语言之界 |
| 智聊（AI 自动聊天获客） | 沟通与成交之界 |

> 一句话统领全线：**无界，让沟通不再有边界。**

### 曾考虑的备选（存档）

越界 BEYOND（锋芒进取）/ 寰界 OMNIA（平台高端）/ 无垠 INFINIA（辽阔未来）/ 寰擎 NEXA / 曦元 AURION。最终选定**无界 BOUNDLESS**。

> ⚠️ 落地前需做**商标 / 域名 / 社媒账号**可用性核查（Boundless 为通用词，建议配合后缀如 `BoundlessAI` / `Boundless.ai` 注册）。

## 三、子产品目录（5 条产品线 · 无界品牌族）

> 命名规则：**中文“双字 + 通俗易懂”，英文统一 `…X` 系列**（X = 突破边界 / 无限变换），形成系列感、呼应母品牌 BOUNDLESS。
> 每个英文都另给一个**更自解释的备选**，供不同渠道使用。

### 1. 幻颜 · FaceX —— AI 换脸　【打破容貌之界】

- **是什么**：图片 / 视频 AI 换脸，把任意人脸自然替换到目标素材上。
- **中文「幻颜」**：幻=变幻，颜=容颜 →「随心变幻的容颜」，不被一张脸限制。
- **英文「FaceX」**（备选 FaceSwap）：Face + X（变换/无限）。
- **目标用户**：内容创作者、影视后期、电商广告、虚拟形象运营。

### 2. 幻声 · VoiceX —— AI 声音克隆　【打破声音之界】

- **是什么**：克隆指定音色，做 AI 配音 / 朗读 / 语音合成（TTS）。
- **中文「幻声」**：幻=变幻，声=声音 →「随心变幻的声音」，与“幻颜”成系列。
- **英文「VoiceX」**（备选 VoiceClone）：Voice + X。
- **目标用户**：有声内容、配音、虚拟主播、客服语音、个人 IP。

### 3. 幻影 · LiveX —— 实时直播换脸 + 换声　【打破身份之界】

- **是什么**：直播 / 连麦 / 视频通话中，**实时**换脸 + 实时变声，低延迟。
- **中文「幻影」**：幻=变幻，影=影像/分身 →「实时百变的分身」，直播里换一个身份登场。
- **英文「LiveX」**（备选 LiveMorph）：Live（实时）+ X（变换），“实时变身”。
- **目标用户**：直播带货、秀场、跨境连麦、需形象/声音保护的主播。

### 4. 通译 · LingoX —— 实时换语言 / 同声翻译　【打破语言之界】

- **是什么**：语音 + 文字的**实时跨语言互译**，双方各说母语也能无障碍沟通（直播实时字幕/配音、跨语言聊天）。
- **中文「通译」**：通=畅通无碍，译=翻译 →「畅通无阻的翻译」，让语言不再是界。
- **英文「LingoX」**（备选 LiveLingo）：Lingo（语言/口语）+ X。
- **目标用户**：跨境直播、跨境电商客服、外贸沟通、国际社群。

### 5. 智聊 · ChatX —— 聚合智能聊天 + AI 自动获客成交　【打破沟通与成交之界】

- **是什么**：聚合多平台（Telegram / LINE / Messenger / WhatsApp）会话，**AI 全程自动聊天、开发客户、推进成交**，可人机协同接管。
- **中文「智聊」**：智=智能，聊=聊天 →「会自动开发客户的智能聊天」。
- **英文「ChatX」**（备选 ChatHub）：Chat + X（突破人力边界，自动成交）。
- **目标用户**：跨境营销团队、私域运营、出海获客、客服外包。

## 四、命名体系逻辑

| 层级 | 取向 | 例子 |
|------|------|------|
| 总公司 | 大气、国际化、命中“无审查自由部署”卖点 | 无界 BOUNDLESS |
| 子产品中文 | 通俗双字、呼应“变幻/无界” | 幻颜·幻声·幻影 / 通译·智聊 |
| 子产品英文 | 统一 `…X` 系列（X=突破边界） | FaceX·VoiceX·LiveX·LingoX·ChatX |
| 英文备选 | 更自解释，给特定渠道用 | FaceSwap·VoiceClone·LiveMorph·LiveLingo·ChatHub |

> 公司名撑场面（对投资人/大客户），产品名好传播（对终端用户/渠道），系列后缀强化品牌族认知。

## 六、官网替换范围与执行计划（待拍板后执行）

- **核心单点**：`website/lib/brand.ts`（`BRAND` 常量，品牌唯一事实源）。
- **硬编码散落**（需逐一改）：`website/lib/content.ts`、`legal-content.ts`、`bot-knowledge.ts`、`daily-content.ts`、`catalog-posts.ts`、`telegram-bot.ts`、`tg-broadcast.ts`，以及 `components/Navbar.tsx`、`Footer.tsx`、`BrandMark.tsx`、各 `app/**/page.tsx`、`manifest.ts`、`opengraph-image.tsx`、`layout.tsx` 等（共约 40+ 文件含旧品牌串）。
- **产品结构变更**：旧站是「2 产品线（华影/灵犀）」，新结构是「5 子产品」，小程序视图（`liveavatar`/`soulsync`）与营销文案需**重构**，不只是改名。
- **建议执行顺序**：① 定稿 5 产品名 → ② 改 `brand.ts` 并扩展为 5 产品 → ③ 全局替换硬编码 → ④ 重构小程序视图/营销文案 → ⑤ 折扣码前缀 / localStorage key / 域名同步。
